# CodeContext worker SDK for Python

Dependency-free Content-Length framing, a streaming message decoder, and protocol
method constants for implementing a CodeContext language worker.

The authoritative wire contract is the
[parser protocol schema](../../src/CodeContext.Parser.Protocol/protocol/parser-protocol.schema.json),
shipped as `protocol/parser-protocol.schema.json` in a CodeContext release. See the
[language-worker SDK guide](../../docs/language-worker-sdk.md) for lifecycle and
graph-ownership rules.
