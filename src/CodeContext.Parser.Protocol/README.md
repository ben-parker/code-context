# CodeContext worker protocol (.NET)

This package provides the versioned parser-worker contracts, Content-Length framing,
JSON-RPC connection, manifest model, and source-generated serializers used by
CodeContext. The canonical cross-language schema is packaged under
`protocol/parser-protocol.schema.json`.

See `docs/language-worker-sdk.md` in the CodeContext repository for the worker
lifecycle, ownership rules, discovery precedence, and native-tree guidance.
